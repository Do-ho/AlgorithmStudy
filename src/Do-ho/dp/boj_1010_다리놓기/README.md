# 다리 놓기

f(1, 1) = 1
f(1, 2) = 2
f(1, 3) = 3
~~

f(2, 2) = f(1, 1)
f(2, 3) = f(1, 2) + f(1, 1)
f(2, 4) = f(1, 3) + f(1, 2) + f(1, 1) = f(1, 3) + f(2, 3)
f(3, 3) = f(2, 2)
f(3, 4) = f(2, 3) + f(2, 2)
f(3, 5) = f(2, 4) + f(2, 3) + f(2, 2) = f(2, 4) + f(3, 4)

따라서 점화식

f(n, k) = f(n-1, k-1) + f(n, k-1)
        = f(n, k-1) + f(n-1, k-1)
