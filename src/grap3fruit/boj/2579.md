## DP

dp[1], dp[2], dp[3] 베이스로 구하고

```
dp[n] = max(

dp[n-2] + a[n] ,

dp[n-3] + a[n-1] + a[n]

 )
```

a[n]번째를 밟기 위해선 아래 두가지 케이스가 와야한다.
a[n-2] → 0 → a[n] [n-2]번째를 밟는 경우.
a[n-3] → 0 → a[n-1] → a[n] [n-3]번째를 밟는 경우.

이 두 경우중 하나가 a[n]번째의 경우의 수로써 가능하고,
이때 최소값을 구하면 된다.
따라서 점화식이 위와 같이 된다.
