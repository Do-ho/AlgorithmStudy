## DP

base dp 를 1,2,3으로 두고

```
4 = 1+3
= 2+2
= 3+1

dp[4] = 1 + dp[3]
= 2 + dp[2]
= 3 + dp[1]

5 = 1+4
= 2+3
= 3+2

dp[5] = 1 + dp[4]
= 2 + dp[3]
= 3 + dp[2]

6 = 1+5
= 2+4
= 3+3

dp[n] = 1 + dp[n-1]
= 2 + dp[n-2]
= 3 + dp[n-3]
```

1,2,3은 값이 아니라, 경우의 수 이다.  
1 + dp[n-1] 이 되는 경우의 수,  
2 + dp[n-2] 이 되는 경우의 수,  
3 + dp[n-3] 이 되는 경우의 수  
이 3 종류 경우의 수가 합해서 dp[n]이 된다.

따라서 dp[n] = dp[n-1] + dp[n-2] + dp[n-3]으로 점화식을 만들 수 있다.
